# Mind JSON Server

## Overview

Mind's JSON server API.

## Good to know

For the following npm script:
```json
"start": "json-server --watch db.json --host 0.0.0.0 --port 4000"
```

`--host 0.0.0.0` is used to ensure json-server listens to all interfaces from the container.
